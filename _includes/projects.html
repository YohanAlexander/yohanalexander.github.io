Z2n Periodogram

The Z2n Periodogram is an open source software for statistical fitting of astronomical data in Python. The software was built following design standards and PEP8 standards, with a command line interface to facilitate community use. In addition, it was packaged and made available in the central PyPI repository for easy installation by the scientific community through package managers such as PIP. Its documentation was built using the Sphinx framework and hosted on ReadTheDocs.

Github

Simulated Pulsar

The Simulated Pulsar is an open source software in Python, which makes use of serial communication via Arduino to visualize data obtained by a luminosity sensor. The obtained signal is read, stored and processed in real time with the use of libraries for the graphical plot of a light curve, that is, luminous intensity as a function of time, in an interactive interface.

Github

The Poker game is an open-source, command-line game that was developed to explore and create technical skills, using the object-oriented paradigm of programming languages. The game was implemented using the characteristics that are the foundations of the paradigm: abstraction, encapsulation, inheritance and polymorphism.

Periodograma Z2n

O Periodograma Z2n é um Software de código aberto para fitting estatístico de dados astronômicos em Python. O software foi construído seguindo padrões de projeto e as normas da PEP8, com uma interface em linha de comando para facilitar o uso da comunidade. Além disso o mesmo foi empacotado e disponibilizado no repositório central PyPI para fácil instalação da comunidade científica por meio de gerenciadores de pacotes como PIP. Sua documentação foi construída com o uso do framework Sphinx e hospedada no ReadTheDocs.

Github

Pulsar Simulado

O Pulsar Simulado é um Software de código aberto em Python, que faz uso de comunicação serial via Arduino para visualização de dados obtidos por um sensor de luminosidade. O sinal obtido é lido, armazenado e processado em tempo real com uso de bibliotecas para a plotagem gráfica de uma curva de luz, ou seja intensidade luminosa em função do tempo, em uma interface interativa.

Github
